    • SECS / GEM

      IOT 기반의 데이터를 실시간 수집하여

    • IOT Platform

      FAB 내에 각종 센서와 단말기 등을 연결하여
      다양한 서비스를 개발하고 운영할 수 있도록 지원하는 시스템입니다.

    SOLUTION

    What is a IO Scalar?

    IO SCALAR

    FAB 안에 발생되는 모든 IO 데이터를 수집하는 제이원소프트의 솔루션입니다.
    수년간 축적된 Know-How를 통해서 고객사에 최적화되고
    안정적인 Service를 제공하며 현장에서 사용하는 대다수의
    FieldBus I/F 를 제공합니다.
